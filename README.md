# MJBConfigTweaks
Core Arma config tweaks for MJB Arma.

## Usage

To use this mod:

1. Clone the repository
2. Pack the `addons/MJBConfigTweaks` directory into a PBO.
3. Add it as a local mod to Arma 3.
